**项目目标**
- 建立一套按 `科目分类` 的 Java 后端八股体系，方便逐科推进，而不是零散地背问题。
- 参考 [小林 coding](https://www.xiaolincoding.com/) 与它的 [Java 面试题汇总](https://www.xiaolincoding.com/interview/) 结构，把高频模块先搭起来，后续再逐步填充细节。

**复习原则**
- 先按科目分类，再按高频程度推进
- 先理解原理和联系，再整理成自己的表达
- 每一科尽量形成“能讲 3-5 分钟”的答案骨架

**科目分类**
- Java 基础
  - 语法基础、面向对象、异常、泛型、反射
- Java 集合
  - List / Set / Map、HashMap、ConcurrentHashMap、底层结构
- Java 并发
  - 线程、线程池、锁、volatile、synchronized、并发容器
- JVM
  - 内存结构、类加载、垃圾回收、常见调优思路
- Spring / Spring Boot
  - IoC、AOP、事务、自动装配、常见注解与启动流程
- MySQL
  - 索引、事务、MVCC、锁、日志、执行计划
- Redis
  - 数据结构、持久化、缓存击穿/穿透/雪崩、高可用
- 计算机网络
  - TCP、HTTP/HTTPS、Cookie/Session、常见网络问题
- 操作系统
  - 进程线程、内存管理、IO、多路复用、死锁
- 消息队列
  - 使用场景、削峰填谷、可靠性、重复消费、顺序消费
- 分布式 / 系统设计
  - 分布式基础、一致性、限流、缓存、服务拆分
- 项目表达
  - 项目背景、技术选型、难点、优化点、结果与复盘

**优先级**
- 第一优先级：Java 基础、集合、并发、MySQL、Redis
- 第二优先级：Spring、计算机网络、操作系统
- 第三优先级：消息队列、分布式、系统设计、项目表达
