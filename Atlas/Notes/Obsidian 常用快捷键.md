## 基础操作

| 快捷键        | 功能                         |
| ---------- | -------------------------- |
| `Ctrl + O` | 快速打开/搜索笔记                  |
| `Ctrl + N` | 新建笔记                       |
| `Ctrl + P` | 命令面板                       |
| `Ctrl + S` | 保存（Obsidian 自动保存，但也保留此快捷键） |
| `Ctrl + W` | 关闭当前面板                     |
| `Ctrl + E` | 切换 编辑/阅读 模式                |
| `Ctrl + ,` | 打开设置                       |

## 编辑相关

| 快捷键                | 功能       |
| ------------------ | -------- |
| `Ctrl + B`         | 加粗       |
| `Ctrl + I`         | 斜体       |
| `Ctrl + K`         | 插入链接     |
| `Ctrl + ]`         | 增加缩进     |
| `Ctrl + [`         | 减少缩进     |
| `Ctrl + D`         | 删除当前段落   |
| `Ctrl + Z`         | 撤销       |
| `Ctrl + Shift + Z` | 重做       |
| `Tab`              | 缩进（在列表中） |
| `Shift + Tab`      | 取消缩进     |

## 导航与面板

| 快捷键 | 功能 |
|---|---|
| `Ctrl + Shift + E` | 切换文件管理器 |
| `Ctrl + Shift + F` | 全局搜索 |
| `Ctrl + Shift + G` | 切换图谱视图 |
| `Ctrl + Shift + C` | 切换反向链接面板 |
| `Ctrl + Shift + M` | 打开大纲 |
| `Ctrl + G` | 切换标签面板 |
| `Ctrl + 1~9` | 切换到第 1~9 个标签页 |

## 面板与窗口

| 快捷键 | 功能 |
|---|---|
| `Ctrl + T` | 新建标签页 |
| `Ctrl + Shift + T` | 恢复最近关闭的标签页 |
| `Ctrl + \` | 拆分面板（水平） |
| `Ctrl + Alt + →` | 将面板向右移动 |
| `Ctrl + Alt + ←` | 将面板向左移动 |
| `Alt + →` | 前进（历史记录） |
| `Alt + ←` | 后退（历史记录） |

## 搜索与替换

| 快捷键 | 功能 |
|---|---|
| `Ctrl + F` | 当前文件内搜索 |
| `Ctrl + H` | 当前文件内替换 |
| `Ctrl + Shift + F` | 全局搜索 |

## 标题跳转

| 快捷键 | 功能 |
|---|---|
| `Ctrl + ↓` | 跳到下一个标题 |
| `Ctrl + ↑` | 跳到上一个标题 |

## 常用 Markdown 语法

| 语法              | 效果          |
| --------------- | ----------- |
| `[[笔记名]]`       | 双链，链接到另一条笔记 |
| `![[笔记名]]`      | 嵌入另一条笔记内容   |
| `[[笔记名\|显示文字]]` | 带别名的双链      |
| `#标签`           | 添加标签        |
| `- [ ]`         | 待办事项        |
| `- [x]`         | 已完成事项       |
| `> [!info]`     | 信息提示框       |
| `> [!tip]`      | 提示框         |
| `> [!warning]`  | 警告框         |
| `` `代码` ``      | 行内代码        |
| `---`           | 分割线         |

> [!tip] 提示
> 所有快捷键都可以在 **设置 → 快捷键** 中自定义修改。
