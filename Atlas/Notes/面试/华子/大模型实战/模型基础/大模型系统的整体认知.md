# 第一阶段（现在最重要）

# Transformer 推理核心链路

这是所有东西的根。

你现在一定要真正搞懂：

---

## 1. Token 是什么

比如：

```
你好世界
```

不是直接输入模型。

而是：

```
15120 3837 9910 ...
```

模型永远只认识：

# token id

---

## 2. Embedding 是什么

token id：

```
15120
```

会变成：

一个高维向量：

x∈Rdx\in\mathbb{R}^dx∈Rd

这一步：

叫：

# embedding

本质：

> “把文字映射到向量空间”。

---

## 3. Attention 是什么（最重要）

这是 Transformer 核心。

一定要理解：

为什么：

```
“它”
```

能知道指代谁。

核心：
$$
\mathrm{Attention}(Q,K,V)=\mathrm{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V
$$

你现在不用硬背。

但一定要知道：

它本质是在：

# 计算 token 与 token 的关系强度。

---

## 4. 为什么会有 KV Cache

这个直接关系：

- 推理速度
- 显存
- 长上下文

这个是：

大模型工程里极其核心的概念。

---

## 5. 下一 Token 预测

模型本质不是“思考”。

而是：

# 预测下一个 token 概率。

例如：

```
中国的首都是
```

模型会算：

```
北京：0.92 上海：0.03 广州：0.01
```
