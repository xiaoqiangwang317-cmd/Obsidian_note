## 1. 介绍

> [! Note] 存储的值
> 可以是字符串、整数或浮点数
## 2. 内部实现


String 类型的底层的数据结构实现主要是 int 和 [[SDS]]（简单动态字符串）


> [!NOTE] 读写能力
> - 对整个字符串或者字符串的一部分进行操作；
> - 对整数或浮点数进行自增或自减操作


> [!NOTE] 应用场景
> - **缓存对象**
> 对于java来说先把对象序列化为JSON，再用Set存入Redis
> - **常规计数**
> - **分布式锁**
> SET lock:order uuid123 NX EX 30
> NX：仅当key不存在时才设置成功
> EX 30: 30s后过期 
> - **共享 session 信息**
